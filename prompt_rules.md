# QUY TẮC LÀM VIỆC (SYSTEM RULES)

1. TỐI ƯU TOKEN: Tuyệt đối không đọc toàn bộ thư mục. Luôn dùng lệnh tìm kiếm (grep, find) để xác định file cần thiết trước khi đọc.
2. TÔN TRỌNG IGNORE: Luôn bỏ qua các thư mục và file được khai báo trong `.antigravityignore`.
3. SỬ DỤNG TOOL: Luôn ưu tiên gọi các công cụ MCP (như Puppeteer) hoặc các Python script đã có sẵn trong project thay vì tự viết lại code từ đầu.
4. KHÔNG ĐOÁN MÒ: Nếu thiếu thông tin hoặc gặp lỗi môi trường, hãy dừng lại và hỏi tôi. Không tự động thử đi thử lại (loop) mù quáng.
